# Learn Python — Week 1 Practice

Each file covers one concept with explanation + 1-2 example programs.
Run them one by one, then modify the code to practice.

## How to run
```bash
cd learn-python
python3 01_basics.py
```

## Files
1. `01_basics.py` — Variables, data types, operators
2. `02_collections.py` — Lists, tuples, sets, dicts
3. `03_control_flow.py` — if/else, loops, comprehensions
4. `04_functions.py` — Functions, *args, **kwargs, lambda
5. `05_exceptions.py` — try/except, custom exceptions
6. `06_file_io.py` — Reading/writing files
7. `07_oop_basics.py` — Classes, objects, __init__
8. `08_oop_inheritance.py` — Inheritance, super(), dunder methods
9. `09_dataclasses.py` — @dataclass, type hints
10. `10_async.py` — async/await, asyncio
11. `11_pydantic.py` — Data validation (critical for AI)
12. `12_decorators.py` — Decorators & generators

## Practice tips
- Type the code yourself — don't copy-paste
- Modify values, break things, see errors
- After each file, build a small script using that concept
